## Tech Stack

- React 19 + TypeScript
- Vite with SWC
- Tailwind CSS v4
- React Router
- Lucide React (icons)

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview

```
